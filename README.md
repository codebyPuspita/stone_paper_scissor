# Stone Paper Scissor Game (Python)

This is a simple command-line based Stone Paper Scissor game written in Python.
The user plays against the computer, which makes a random choice.

## How the Game Works
- The computer randomly chooses Stone, Paper, or Scissor.
- The user inputs their choice using:
  - s → Stone
  - p → Paper
  - x → Scissor
- The program compares both choices and declares the result.

## How to Run the Program
1. Make sure Python is installed on your system.
2. Save the file as `game.py`.
3. Open terminal or command prompt.
4. Run the command:
   python game.py

## Rules
- Stone beats Scissor
- Scissor beats Paper
- Paper beats Stone
- Same choice results in a Draw

## Features
- Random computer choice using `random` module
- Input validation for wrong choices
- Clear output showing both player and computer choice

## Sample Input
Enter s for stone, p for paper and x for scissor: s

## Sample Output
You chose stone  
Computer chose paper  
You lose
